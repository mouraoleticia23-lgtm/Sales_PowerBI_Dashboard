# Sales_PowerBI_Dashboard
The dashboard presents a comprehensive sales analysis between 2017 and 2019, focusing on revenue, targets, salesperson performance, and payment methods. The report allows to track the evolution of results over time and identify performance patterns.

Key analyses included: 
    • Monitoring of annual targets and actual revenue, 
    • Revenue comparison by year and by month, 
    • Sales analysis by payment type, 
    • Indicators of invoiced vs. canceled sales, 
    • Individual salesperson performance, 
    • Detailed table summarizing sales

The dashboard was developed in Power BI, with a interactive look, allowing dynamic filters to support strategic decision-making.
